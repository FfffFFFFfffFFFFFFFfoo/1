# Структура:
```text
 smart-home/
 ├── index.html              # Главная страница (base.html)
 ├── help.html               # Страница помощи
 ├── light.html              # Управление светом
 ├── security.html           # Управление безопасностью
 ├── temperature.html        # Управление климатом
 ├── energy.html             # Управление энергией
 ├── css/
 │   └── custom.css          # Пользовательские стили
 └── js/
     └── scripts.js          # Общие скрипты
```

# Базовые задачи:
- base.html - главная страница сайта, где будут расположены ссылки на остальные страницы с функциями
- help.html - страница-помощник для пользователя, где будут инструкции ко всем функциям
- light.html - страница с функционалом управления светом в доме
- security.html - страница с функционалом управления охранной системой
- temperature.html - страница с функционалом управления температурой и климатов дома
- energy.html - страница с функционалом управления энергией в доме
